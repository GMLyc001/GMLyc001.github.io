变量、常量，作用域

Created2023-03-11|Updated2023-03-11
|Post View:

j

变量

  1. Java是强类型语言，每个变量必须声明类型

  2. Java变量要包含变量名，变量类型和作用域

    变量作用域

    三种，类变量，实例变量，局部变量

    class 类当中还可以定义一些属性（变量），还可以定义其他方法除了main方法。

    1. 局部变量，放在main方法里的变量，声明周期就存在于main方法的花括号里

      而且使用前必须初始化，例如int a=10；

      不初始化就会报错。

    2. 实例变量，定义在方法的外面，class类的里面。从属于对象（类），且可以不初始化。

      不初始化的话，默认初始化为默认值，例如0。

      Boolean默认是false，除了默认值基本类型，默认都是null。

​ 3. 类变量，加static的。

常量

  1. 常量设定后不允许被修改。

  2. 常量一般用大写字符表示

    例如：static final double PI=3.14

​ 结果一样 final static double PI=3.14

变量的命名规则

  1. 所有的变量，方法，类名，要见名知意
  2. 类成员变量：首字母小写和驼峰原则：monthSalary，除了第一个单词外，后面的首字母大写
  3. 局部变量：首字母小写和驼峰原则。
  4. 常量：大写字母和驼峰原则：MAX_VALUE
  5. 类名：首字母大写和驼峰原则：Man，GoodMan
  6. 方法名：首字母小写和驼峰原则：run（），runRun（）

养成良好的代码书写习惯，遵守规则
